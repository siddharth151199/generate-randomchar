# hexclane/randomchar
This library is for front-end devlopment used in React js

## Installation
using npm

`npm i generate-randomchar`

In React.js

```javascript
import React from 'react'
import RandomChar from 'generate-randomchar';

export default function App() {
  return (
    <div>
      <RandomChar name="example" />
    </div>
  )
}
```
